# Landing Page - ONG Elos Reciclagem

## 📋 Descrição

Landing page completa para a ONG Elos Reciclagem, focada em educação sustentável para crianças e apresentação do jogo educativo "Super Coletor".

## 📁 Estrutura dos Arquivos

```
export/
├── index.html      # Estrutura HTML da página
├── styles.css      # Estilos CSS completos
├── script.js       # Funcionalidades JavaScript
└── README.md       # Este arquivo
```

## 🚀 Como Usar

1. **Abrir Localmente**
   - Baixe todos os arquivos na mesma pasta
   - Abra o arquivo `index.html` em seu navegador
   - Não é necessário servidor local

2. **Hospedagem**
   - Faça upload dos arquivos para seu servidor web
   - Certifique-se de manter a estrutura de pastas
   - Os arquivos são estáticos e não requerem backend

3. **Customização**
   - **Cores**: Edite as variáveis CSS em `:root` no arquivo `styles.css`
   - **Textos**: Edite diretamente no arquivo `index.html`
   - **Imagens**: Substitua as URLs das imagens no `index.html`
   - **Funcionalidades**: Modifique o arquivo `script.js`

## 🖼️ Imagens

As imagens utilizadas são provenientes do Unsplash:
- Hero: Crianças aprendendo sobre reciclagem
- Sobre: Equipe de voluntários
- Comunidade: Crianças plantando árvores
- Jogo: Tela colorida de tablet

Para substituir:
1. Localize as tags `<img>` no HTML
2. Substitua o atributo `src` pela nova URL da imagem
3. Mantenha o atributo `alt` para acessibilidade
